Begins by getting a list of columns and iterating over that list. For each column, we determine whether validation should occur. We then split the @validate metadata into a list of validation conditions. If a condition has arguments, we extract the arguments. We then run each validation method on the column data and generate error messages for those validation conditions that failed.
